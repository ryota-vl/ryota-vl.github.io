CKA/CKAD 2023 考試心得

2023 年 01 月 03 日 ◦ 5 分鐘 ◦

前言

我們團隊一直都有在用託管的 kubernetes 服務，像是 Amazon EKS、Azure AKS 部屬應用程式或是服務，對於 kubernetes 的基本基礎和元件都不是很熟悉，想透過學習的方式，有個全面性的概念與知識。

考試介紹

The Linux Foundation 推出的 Kubernetes 證照，目前共有四張，分別是

  • Certified Kubernetes and Cloud Native Associate (KCNA)
  • Certified Kubernetes Application Developer (CKAD)
  • Certified Kubernetes Administrator (CKA)
  • Certified Kubernetes Security Specialist (CKS)

購買考試

我是在 2021 的黑色星期五，用 35% 的折扣碼，購買原價 375 美金，實際價格是 243.75 美金，2022 年價格原價 395 美金，購買後一年期限內，都可以從系統上預約考試。

CKA 考試範圍

  • Cluster Architecture, Installation & Configuration - 25%
  • Workloads & Scheduling - 15%
  • Services & Networking - 20%
  • Storage - 10%
  • Troubleshooting - 30%

CKAD 考試範圍

  • Application Design and Build - 20%
  • Application Deployment - 20%
  • Application Environment, Configuration, and Security-25%
  • Services and Networking - 20%

雖然考前網路上的文章都說兩邊的內容重複，但實際考下來，CKA 比較偏向維運的範圍，而 CKAD 則是偏向開發佈署的範圍。像是 CKA 就會考到 Node 的更新，而 CKAD 則是會考到佈署流程或是如何分散佈署時的更新。

學習資源

  • Certified Kubernetes Administrator (CKA) with Practice Tests
  • Kubernetes Certified Application Developer (CKAD) with Tests
  • killer.sh (模擬考題): 記得從 The Linux Foundation 的入口進去，才會得到兩次免費模擬考的機會。

模擬考題

記得從 Linux Foundation 報名，會有兩次模擬考的機會，總共會有 preview questions、mock questions、Extra questions 大概有 25-26 題左右。

考試環境

  • 視訊用攝影機: 用筆電的攝影鏡頭就可以了，但就是要一直調整角度給監考官，如果看不清楚的話，考官也會叫你重新照給他看。
  • CCTV(監視器): 因為家裡沒有一個整齊且安靜的空間，這幾次考試都是租借小樹屋，所以房間內都有監視器，這部份監考官會希望你換一間考試的房間或是面向攝影機考試。如果有一個獨立的空間或是利用公司的會議室是最好的選擇。

考試前

第一次考試的時候，監考官認為護照的名字和系統的名字不一樣，本來是打算不讓我考試，後來發現在系統裡面發現是名字之間的連字符號(-)，系統上沒填到，馬上進入系統更新，重新進入 PSI 系統裡面，等待監考官檢查考試環境和證件，當時應該滿多人考試，等待時間很長，大約半小時，也是剛好提醒監考官系統名字有更新，才能夠順利考試。

(!) 名字一定要跟護照一模一樣(包含連字號)，其他證件基本上沒有英文名字是不能證明，信用卡也是不行，這點千萬要注意。

考試技巧

  • 螢幕能夠越大越好，基本上題目和參考資料顯示各一半，所以文件查找或是尋找 keyword 都有點困難。
  • 基本上沒有時間每題都查文件，能夠用 kubectl 產生就使用 kubectl 產生。

考試後

大概 24 小時之後，可以從網站查詢到成績，也會寄信通知。
